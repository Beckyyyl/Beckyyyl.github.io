遇到的问题

deploy不成功

  • 1 deploy not found: git
    在 Git Bash 页面 输入 npm install hexo-deployer-git –save即可
  • 2 deploy时出现： Permission to…… denied to ……
    这是因为同一台电脑里有两个或以上的账户，这个时候应该通过配置私钥来解决。
    然而 由于我在配置私钥过程中出现了一些问题，导致方法不奏效，于是，果断选择了重装系统。
    于是我明白了一个道理：
    重装系统能解决大部分的电脑问题。（手动滑稽）
